## Release Notes

~~~markdown
Initial release of Pixiv Toolbox.

Highlights:

- Browser-based Pixiv OAuth refresh token helper
- Pixiv API connectivity test workflow
- Tag search with filters for sorting, date range, ranking mode, and minimum bookmarks
- Gallery and list preview modes
- Original, large, and medium image download options
- Local proxy detection for common proxy ports

Install:

```bash
pip install -r requirements.txt
playwright install chromium
python app.py
```
~~~
